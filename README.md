# 📊 Sales Dashboard (Power BI)

## 👀 About this project

This is a simple **Sales Dashboard** I built using Power BI to understand how a business is performing across different regions, products, and teams.

I created this project mainly to practice:

* Data cleaning
* Data modeling
* Building interactive dashboards

---

## 🔥 What I built

In this dashboard, you can see:

* Profit split across regions (East, West, North, South)
* Category-wise profit (like Accessories, Electronics)
* Sales comparison between regions
* Team-wise contribution using a waterfall chart
* KPI cards showing total customers, sales, and profit

Also added filters for:

* Product Name
* Region

So you can interact and explore the data easily.

---

## 🗂️ Data used

I used a sample sales dataset with:

* Customers
* Products
* Sales data
* Teams
* Segments

---

## 🛠️ Tools used

* Power BI
* DAX (basic calculations)

---

## 📊 Key observations

* East region is giving slightly higher sales
* Accessories category has more profit compared to others
* Team performance is gradually increasing

---

## 🎯 Why I did this

I wanted to understand how real business dashboards are built and how to convert raw data into useful insights.
